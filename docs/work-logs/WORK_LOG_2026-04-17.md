\#\#\# Problem 3: DNS replacement during custom domain setup  
\- Important note:  
  Cloudflare showed that the existing root DNS record would be replaced by a Pages target.  
\- Lesson:  
  Understand that domain activation may modify the current DNS mapping, not just add a new record.
