# SPP Foundry

This repository contains the main landing page for SPP Foundry.

## Live site
https://sppfoundry.co/

## What this is
SPP Foundry is the main brand site for showcasing small AI products, MVP builds, experiments, and future client-facing work.

This site is intended to become the central entry point for:
- product showcase
- AI MVP case studies
- service positioning
- future monetization pages
- contact / lead generation

## Deployment
This site is deployed with Cloudflare Pages.

- GitHub repo: `sylajj0927/sppfoundry-landing`
- Production branch: `main`
- Entry file: `index.html`
